## Overview
Snapgram is a modern social media application built with cutting-edge web technologies, designed to provide a seamless photo-sharing experience.

## 🚀 Tech Stack
- React
- TypeScript
- Tailwind CSS
- Shadcn/ui
- Vite
- Firebase/Appwrite (backend)

## ✨ Features
- User Authentication
- Photo Uploading
- Social Interactions (Like, Comment, Share)
- User Profiles
- Responsive Design

## 🛠️ Installation

### Prerequisites
- Node.js (v18+)
- npm or yarn

### Setup
1. Clone the repository
\`\`\`bash
git clone https://github.com/Abhayaj247/Snapgram.git
cd snapgram
\`\`\`

2. Install dependencies
\`\`\`bash
npm install
\`\`\`

3. Run the development server
\`\`\`bash
npm run dev
